# Common ARIA attributes and their uses

## aria-label: provides a custom label for an element specially when the element doesn't have visible text.

## aria-labelledby: references the id of another element to provide a label. useful for grouping elements.

## aria-describedby: additional information to the element

## aria-hidden: used for elements hidden for screen readers but visible to sighted users

## aria-live: content updates dynamically.

## role: explicit role for accessibility; alert, dialog

## aria-expanded: used when an element can toggle between expanded and collapsed states.

## aria-controls: to indicate which element is controlled by a button

## aria-current: highlighting the current item in a set.
